Child Pseudo classes - :first-chld, :last-child, :nth-child(n), :nth-last-child(n)

Use :first-child to style the first list/menu item

Use :last-child to style the last list/grip item

Apply :nth-child(n) for alternating rows or repeating patterns

Apply :nth-last-child(n) for end based patterns, reverse alternates
