Inline Elements: it takes only the required space e.g a, span, strong, image, em 1) It does not allows width and heights, 2) it rerspects ONLY Left and Right margins and margin. 3) It does not accepts positioning e.g relative, absolute, sticky, etc. NOTE: To make inline elements behave like a block elements use: display: block;
Block Elements : It takes the size of the parent container. eg h1 - h6, p, form, div, ul, li
1) Allows width and heights,
2) it rerspects all margins and margin.
3) It accepts positioning e.g relative, absolute, sticky, etc. NOTE: To make Block elements behave like an inline element use: display: inline;
