Array Creation Comparison

Array Creation Comparison

Feature Array Literals Array Constructors
Syntax Uses square brackets `[]` Uses the `new` keyword with the `Array()` constructor
Example const fruits = ['apple', 'banana', 'orange']; const fruits = new Array('apple', 'banana', 'orange');
Advantages - Shorter syntax
- Preferred for predefined values		 - Can create arrays dynamically
- Allows specifying initial length
Disadvantages N/A - May behave unexpectedly with single numeric argument
- Inconsistent behavior with multiple arguments

Array Creation Methods Comparison

Method Description Example
Array.of() Creates a new array instance with a variable number of arguments, regardless of the number or type of the arguments. const arr = Array.of(1, 'two', 3, [4, 5]);
Array.from() Creates a new array instance from an array-like or iterable object, with an optional mapping function to modify the elements. Use for cloning array const str = 'hello';
const charArray = Array.from(str);

Array Mutation Methods Comparison

Method Description Example
push() Adds one or more elements to the end of an array and returns the new length of the array. const fruits = ['apple', 'banana'];
fruits.push('orange', 'kiwi');
unshift() Adds one or more elements to the beginning of an array and returns the new length of the array. const fruits = ['apple', 'banana'];
fruits.unshift('orange', 'kiwi');
pop() Removes the last element from an array and returns that element. const fruits = ['apple', 'banana', 'orange'];
const lastFruit = fruits.pop();
shift() Removes the first element from an array and returns that element. const fruits = ['apple', 'banana', 'orange'];
const firstFruit = fruits.shift();
