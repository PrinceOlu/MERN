JavaScript Data Types

Data Types
Data Type Description
Primitive Data Types
  • String: Represents a sequence of characters, enclosed within either single (' ') or double (" ") quotes.
  • Number: Represents both integer and floating-point numbers.
  • Boolean: Represents a logical value, either true or false.
  • Undefined: Represents a variable that has been declared but not assigned a value.
  • Null: Represents the intentional absence of any value.
  • Symbol: Represents a unique identifier.
Non-primitive Data Types (Reference Types)
  • Object: Represents a collection of key-value pairs. Objects are reference types and can hold any combination of primitive or non-primitive data types.
  • Array: Represents a special kind of object used to store lists of elements. Arrays are ordered collections of values.
  • Function: Functions in JavaScript are first-class objects, which means they can be treated like any other variable. Functions can be assigned to variables, passed as arguments to other functions, and returned from other functions.
