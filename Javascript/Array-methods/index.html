Method Use Differences Sample Code
map() Creates a new array by applying a function to each element of the original array. Produces a new array with the same length as the original array. 
const arr = [1, 2, 3, 4];
const doubled = arr.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6, 8]
filter() Creates a new array with all elements that pass the test implemented by the provided function. Produces a new array with fewer elements than the original array, depending on the condition. 
const arr = [1, 2, 3, 4];
const evens = arr.filter(num => num % 2 === 0);
console.log(evens); // Output: [2, 4]
reduce() Applies a function against an accumulator and each element in the array to reduce it to a single value. Produces a single output value (such as a number, string, or object) based on the operation. 
const arr = [1, 2, 3, 4];
const sum = arr.reduce((acc, num) => acc + num, 0);
console.log(sum); // Output: 10
